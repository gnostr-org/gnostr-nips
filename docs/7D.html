NIP-7D

Threads

draft optional

A thread is a kind 11 event. Threads SHOULD include a subject with a summary of the thread's topic.

{
  "kind": 11,
  "content": "Good morning",
  "tags": [
    ["subject", "GM"]
  ]
}

Replies to kind 11 MUST use NIP-22 kind 1111 comments. Replies should always be to the root kind 11 to avoid arbitrarily nested reply hierarchies.

{
  "kind": 1111,
  "content": "Cool beans",
  "tags": [
    ["K", "11"],
    ["E", <event-id>, <relay-url>, <pubkey>]
  ]
}
